# ai-elements-mui

Material UI flavored React components for building AI chat interfaces.

## Installation

```bash
npm i ai-elements-mui
```

## Usage

Import components via subpath exports:

```ts
import { Message, MessageContent } from "ai-elements-mui/message";
```

Or import from the root barrel:

```ts
import { Message, MessageContent } from "ai-elements-mui";
```
